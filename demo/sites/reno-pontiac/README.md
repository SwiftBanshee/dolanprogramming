# Reno Pontiac Reno — Website

Static, mobile-friendly website with:
- Sticky header + hamburger menu
- Animated modern sections (“moving parts”)
- Full EN/FR language toggle (defaults to English)
- Local Quebec SEO + LocalBusiness schema
- Separate photo-first gallery page (`gallery.html`)
- Footer credit: “Created by Dolan Programming” (links to `https://www.dolanprogramming.ca`)

## Run locally

- Open `index.html` in your browser, **or**
- Serve the folder with any static server (recommended so anchors + assets behave consistently).

Example (Node):

```bash
npx serve .
```

## Customize

- **Services / wording**: edit `index.html` and/or the translation strings in `app.js`
- **Colors / motion**: edit `styles.css`
- **Facebook link**: already set to the provided page
- **Gallery photos**: put images in `assets/gallery/` and update the file names in `gallery.html`
